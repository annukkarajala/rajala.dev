# [rajala.dev](https://rajala.dev)

> Personal portfolio. Built with [Next.js](https://nextjs.org/) and deployed on [Vercel](https://vercel.com).

```
$ npm install
```

## Development
```
$ npm run dev
```
Open http://localhost:3000 in your browser.
